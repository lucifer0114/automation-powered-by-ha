# Comment Tone and Rollback Notes

## Purpose
Use this when a Weibo comment needs to be corrected for topic fit or tone.

## Lessons from this session

- Always re-read the post text before generating the comment.
- Match the comment to the post theme, not to a previous URL or previous topic.
- If the user says the comment is off-topic, treat it as a content correction, not just a wording tweak.
- Prefer a short, ordinary-user tone when requested:
  - more casual than media copy
  - less polished than a summary
  - natural phrases like "挺有意思", "挺接地气", "真的会宠游客"
- Before reposting, delete the incorrect comment if it was already published.

## Rollback sequence

1. Open the post in the persistent profile.
2. Switch comments to `按时间`.
3. Find the most recent matching self-comment.
4. Click the delete entry in the comment item.
5. Confirm deletion in the dialog.
6. Verify the old comment text is gone.
7. Post the corrected comment.
8. Re-capture evidence.

## Tone ladder

From more formal to more casual:
- `活动很有温度，孩子们可以近距离接触特警车。`
- `这活动挺有意思，孩子能近距离看看特警车。`
- `这活动挺有意思，孩子能近距离看看特警车，感觉挺新鲜也挺接地气的。`

## Pitfalls

- Reusing a comment from another post because it sounded “good”.
- Writing too polished or summary-like when the user asked for a plain网友口吻.
- Leaving the wrong comment up after posting a corrected one.
